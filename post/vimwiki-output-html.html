Vimwiki와 html 블로그 통합하기

Vimwiki를 사용하고 있는데 html 블로그를 함께 관리하려니 쉽지 않았다. 어떻게하면 잘 사용할 수 있을까 많이 고민했는데 Vimwiki에 auto_export라는 설정이 있는 것을 발견했다. 이 설정을 켜놓으면 저장할 때마다 지정한 디렉토리에 자동으로 export할 수 있다. Vimwiki에서 작성한 내용이 자동으로 블로그에 저장되도록 할 수 있는 것이다.

현재 .vimrc에 저장한 Vimwiki 설정은 아래와 같다. 

let g:vimwiki_list = [
    \{
    \   'path': '~/wiki/',
    \   'ext' : '.md',
    \   'diary_rel_path': '.',
	\   'template_path': '~/wiki/templates',
	\   'template_default': 'default',
	\   'template_ext': '.html',
	\   'path_html': '~/log/post/',
	\   'auto_export': 1
    \},
\]
