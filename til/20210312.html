VIM normal .

VIM $에서 모든 행의 끝에 세미콜론을 추가하는 작업을 했었는데 오늘 새로운 것을 배웠다. . 명령으로 처리할 수가 있다.


var foo = 1
var foo = 'a'
var foo = 'z'

이런 코드가 있을 경우 기존에는 VG:norm A; 이 작업으로 진행했다. 전체 행이니까 G를 쓸 수 있었지만 범위가 있다면 쉽지 않고 여기저기 흩어져있는 작업이라면 더 쉽지 않다. .을 이용해 쉽게 처리할 수 있다.


var foo = 1;
var foo = 'a';
var foo = 'z';

반복을 수행할 행에서 A; 명령을 실행하고, 범위를 선택한 후 :norm .를 입력하면 된다. 여러 작업을 할 때 :norm [명령]을 입력하는 게 귀찮았는데 :norm .으로 조금 더 간략해졌다.
