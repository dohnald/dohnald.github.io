else if

PSR-12에 따르면 `else if` 대신 `elseif`를 써야한다.

The keyword elseif SHOULD be used instead of else if so that all control keywords look like single words.

 <?php
if ($expr1) {
    // if body
} else if ($expr2) {
    // else if body
} else {
    // else body
}

위 코드를 phpstorm에서 정리하면(reformat code) 아래와 같이 바뀐다.

 <?php
if ($expr1) {
    // if body
} else {
    if ($expr2) {
    // else if body
    } else {
    // else body
    }
}
