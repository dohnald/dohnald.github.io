VIM $

여러 행의 문자가 있을 때 앞뒤에 메시지를 추가할 수 있다. VIM gv에서도 언급했지만 여러 줄을 선택해 앞뒤에 데이터를 넣는 경우가 있을 때는 아래와 같이 작업했다.


var foo = 1
var bar = 'bar'
var foobar = foo + bar

V로 선택하고 norm A; : 행의 맨 끝에 ; 추가


var foo = 1;
var bar = 'bar';
var foobar = foo + bar;

다음과 같이 해도 작동한다. jj$로 선택하고 A; : 행의 맨 끝에 ; 추가


var foo = 1;
var bar = 'bar';
var foobar = foo + bar;

응용하면 이렇게도 쓸 수 있겠다. jj^로 선택하고 I const : 행의 맨 앞에 const 추가


const var foo = 1
const var bar = 'bar'
const var foobar = foo + bar
